免费公共代理(corsproxy/allorigins)会限流/抽风，且 双击打开(file://)时 corsproxy 会直接拒绝 → 抓不到数据。
三种彻底解决（任选）：
① 自建代理（最稳）：5分钟部署一个 Cloudflare Worker（免费·无限流·file://也能用），把地址填上面 → 数据 100% 实时。代码+步骤见 README「自建代理」。
② 本地服务器：文件夹里运行 python3 -m http.server，再访问 localhost:8000。
③ 托管 GitHub Pages（和你 pilot-pipeline 一样）。
